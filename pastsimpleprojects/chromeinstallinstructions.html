Install Chrome on Windows

  1. Use your internet browser to navigate to
  2. Click on the blue “Download Chrome” button.
  3. A new window will pop up. On this new window, you will see a check box with the following text next to it “Help make Google
    Chrome better by automatically sending usage statistics and crash reports to Google. Learn more”
  4. It is suggested that you un-check that check box.
  5. Click the blue “Accept and Install” button.
  6. If prompted, click “Save”.
  7. If you chose “Save”, navigate to the downloaded file, which can normally be found in your “Downloads” folder on your computer.
  8. The file’s name is “ChromeSetup”. Double-click on it to start installing.
  9. A window may pop up asking if you want to allow Google Chrome to make changes to your PC. To continue, click the “Yes” button.
  10. Google Chrome will proceed to download and install itself.
  11. Congratulations. Chrome should have successfully been installed on your computer.
    • Windows 7: A Chrome window opens once everything is done.
    • Windows 8 & 8.1: A welcome dialog appears. Click Next to select your default browser.
    • Windows 10: A Chrome window opens after everything is done. You can make Chrome your default browser.

Install Chrome on Mac

  1. Download the installation file from
  2. Open the file called "googlechrome.dmg."
  3. In the window that opens, find Chrome.
  4. Drag Chrome to the Applications folder.
    • You might be asked to enter the admin password.
    • If you don't know the admin password, drag Chrome to a place on your computer where you can make edits, like your desktop.
  5. Open Chrome.
  6. Open Finder.
  7. In the sidebar, to the right of Google Chrome, click Eject.
